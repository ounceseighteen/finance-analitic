# Трекер личных финансов

Простое веб-приложение для учёта доходов и расходов с авторизацией пользователей.

## Быстрый запуск

### Установите зависимости
pip install -r requirements.txt

### Запустите приложение

python app.py

### Откройте в браузере

http://localhost:3000
